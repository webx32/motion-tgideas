Example
 		var sourceArr = []; //需要加载的资源列表
		new mo.Loader(sourceArr,{
			onLoading : function(count,total){
				console.log('onloading:single loaded:',arguments)
			},
			onComplete : function(time){
				console.log('oncomplete:all source loaded:',arguments)
			}
		})

  • 名称 类型 参数 默认值 描述
    res Array <optional>
    		 [] 需要加载的资源列表
    opts object <optional>
    		 配置参数
    Properties
    名称 类型 参数 默认值 描述
    onLoading Function <optional>
    		 当个资源加载完成后的回调
    onComplete Function <optional>
    		 所有资源加载完成后的回调
    loadType Number <optional>
    		 0 0为并行加载 1为串行加载
    minTime Number <optional>
    		 0 加载单个资源需要耗费的最少时间(毫秒)
    dataAttr Strnig <optional>
    		 preload Dom元素需要预加载资源的dom属性默认：data-preload
